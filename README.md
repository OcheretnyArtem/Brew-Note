# Brew-Note
## Android app to simplify baristas work
Application for easy creation, storage and distribution of coffee profiles
### Technologies used:
 * Clean architecture 
 * MVI  
 * Coroutines
 * Compose
 * Hilt
 * Room
 * FireStore
 * DataStore
---
Development of the app takes place in the dev branch. Main branch - for the released version.
